# Pran.ai

A startup project for building AI tools and agents.

## Description

Context-Aware Life Intelligence Ecosystem: Replacing passive tracking with a proactive ecosystem. Utilizing a central Orchestrator to ingest high-stakes life events—Career pivots, Founder transitions, or Job loss—to trigger a coordinated response:

- **The Planner**: Maps multi-horizon goals with next best actions.
- **The Matchmaker**: Scouts human capital (VCs, Recruiters) to accelerate outcomes.
- **The Sentinel**: Manages operational logistics and proactive nudges.

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the main script: `python main.py`

## Features

- Orchestrator for event detection and coordination
- Planner agent for goal mapping and action planning
- Matchmaker agent for networking and human capital scouting
- Sentinel agent for operational management and nudges
